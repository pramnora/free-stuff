    Programming

  • GitHub
    • FREE Code repository, store your codes online...so, that you don't forget...?! They also give your FREE web site space to use, as well.
  • Linode
    • Web server/web site hosting...at Linode. Strictly speaking, Linode isn't FREE(not unless you get a special offer which is, usually, given to first time users); however, if you just wish to 'test' if BASH shell scripting code works/or, practice doing Linux web server admin/Run Apache2 web server software/together with PHP to build web pages...; then, one can simply create a web server on the fly/then, pull it straight back down again after say 1 hour long...the cost is merely pennies.     • The FREE Netlify service, is often used in conjunction with GitHub...by linking to a GitHub source code directory; thus, making the otherwise purely 'static' source code become 'live/interactive'.     • Python, is a FREE programming language...which can be used both offline/online (in the form of DJango/PyScript). It also uses plenty of libraries...such as one can use it to do Logo turtle graphics programming/and, build OOP UI's using TKInter/-etc. It can also be used to do 'web scraping'/-etc. Consequently, Python is one of the most diverse 'general purpose' programming languages ever created with tons of different possible uses.
